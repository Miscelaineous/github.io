Recursion

What is it?

It is a routine that calls itself in a repetitive routine. It's a straight-forward definition but it can become incredibly complex in practice.

When do you use it?

It is often used as a way to iterate through data. As such, it is an alternative to loops and functions that do the same.

It is best used for large tasks that can be broken down into repetitive "sub-tasks" - the "base case" being the smallest size without calling itself.

How do you use it?

You need to take a function, say def count as below, and then call that function within the function (if n>). You must remember to stop calling it (count(n-1)) within the function too!

def count (n):
if n > 0:
print n
count (n-1)
else:
print "Done"
